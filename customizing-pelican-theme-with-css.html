Customizing Pelican theme with CSS

  • Dec 06, 2024
  • Categories: Python
  • Tags: #python

Default themes do not always suit our needs. Sometimes we want to make new designs without creating new themes. Just make a few improvements. But creating your own styles can be tricky. There are two ways.

  1. Make changes in theme's css file. But every time you need make it twice: in progect/theme/static/css/main.css and in /usr/local/lib/python3.11/dist-packages/pelican/themes/yourtheme/static/css/main.css (theme's path in pelicanconf.py). If you have scss files or imported files you need to make changes there. It's complicated.

  2. Copy the theme's main.css project/content/css folder, rename it in custom.css, make changes and write the correct path to the custom CSS file at pelicanconf.py:

SITEURL = "." // Make your custom.css visible from nested folders 
CSS_FILE = 'custom.css' 
STATIC_PATHS = ['images', 'css']
THEME_STATIC_DIR = '' // Point to the static files in the project's root content folder (not the theme's static folder!)
LOAD_CONTENT_CACHE = False
RELATIVE_URLS = True // Use it if your site pages don't render custom CSS file

Put your header pic in the project/content/images folder and write the correct path in your custom.css file:

#container header {
  background-image: url("../images/hacker2.jpg"); 
  background-size: auto;
  background-repeat: no-repeat;
}

The solution is correct if the output folder of your pelican static site has this structure:

OUTPUT folder
-index.html
-css
--custom.css
-images
--header.jpg

Pay attention that you will transfer the OUTPUT folder on yoursite.github.io (not your project's folder!) and the path to the custom CSS file should be relative to the OUTPUT folder structure.

Most users want to change the "Powered by Pelican" link. This link is located mostly in the footer tag of your base.html file. You can delete it manually if your theme is not supported custom footer variable. Don't forget to delete the link also in /usr/local/lib/python3.11/dist-packages/pelican/themes/yourtheme/templates/base.html

Enjoy your new theme!
